🔑 Google OAuth2 Configuration
Google Cloud Console → APIs & Services → Credentials → OAuth 2.0 Client IDs
📋 Setup Guide (5 menit)
1
Buka console.cloud.google.com → New Project "HBD Manager"
2
APIs & Services → Library → Enable: Calendar API, Tasks API, People API, Sheets API
3
OAuth Consent Screen → External → App name "HBD Manager" → Add test users (email kamu)
4
Credentials → Create OAuth 2.0 Client ID → Web application → Authorized origins: URL file HTML ini
✓
Copy Client ID → paste di config → Save → Login Google → ✅ Auto-reminder aktif!
